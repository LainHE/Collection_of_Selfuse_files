# BatchFiles

`refresh.bat` would refresh the network. If you feel the net does not work well, try to use this and reboot the computer.  
`wallPaperExtractor.bat` would extract the Windows Spotlight photos out to `C:\Users\YourAccountName\Pictures\WindowsLockscreenPhoto\`.  
`copy2all.bat` would copy the target file into folders in same diractory.  
`closeScreen.bat` would turn off the monitor(s) without making computer sleep or hibernate. Click || move the mouse, or tap the keyboard to "let there be light".  
`清理V标题视频.bat` move video names containing only v/V to the recycle bin. duplicate name files that Windows auto adds suffixes such as (2) have been considered. Echo text only in Chinese.
